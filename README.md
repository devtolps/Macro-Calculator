Macro Calculator

Aplicação web desenvolvida com HTML5, CSS3 e JavaScript Vanilla para auxiliar no planejamento nutricional. A aplicação calcula macronutrientes personalizados, oferece diferentes modelos dietéticos, permite pesquisar alimentos através da Open Food Facts API e calcula automaticamente os macronutrientes de refeições montadas pelo usuário.

Tecnologias utilizadas
HTML5
CSS3
JavaScript (Vanilla)
Web Storage API
Web Crypto API
Fetch API
Open Food Facts API
Principais funcionalidades
Cálculo personalizado de calorias e macronutrientes.
Seleção entre diferentes modelos de dieta.
Banco de alimentos integrado.
Pesquisa de alimentos em tempo real utilizando a Open Food Facts API.
Montagem de refeições com cálculo automático de proteínas, carboidratos, gorduras e calorias.
Armazenamento local de informações do usuário utilizando Web Storage.
